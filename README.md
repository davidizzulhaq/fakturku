# Fakturku, by UMKM Beres

**Buat invoice, kwitansi, surat penawaran & proforma invoice gratis**, tanpa daftar, tanpa watermark, langsung cetak/PDF.

🔗 **Live**: https://davidizzulhaq.github.io/fakturku/

## Fitur
- Invoice profesional A4 (PPN opsional, terbilang otomatis)
- Kwitansi resmi dengan terbilang Bahasa Indonesia otomatis
- Surat Penawaran (ID) & Quotation (EN), sekali isi, dua bahasa
- Proforma Invoice (EN) untuk ekspor, amount in words otomatis
- Logo usaha sendiri, data tersimpan di perangkat (localStorage, tanpa server)
- 100% client-side, privasi penuh

## Paket Lengkap (berbayar)
Template Excel UMKM: pembukuan kas, stok, piutang, invoice. [Lihat paket](https://davidizzulhaq.github.io/fakturku/paket.html)
